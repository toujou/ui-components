# Toujou Topbutton

Web component for the "top button" element

## Installation

      npm install @toujou/toujou-topbutton

## Usage
```html
 <toujou-topbutton title="Nach oben" aria-label="Nach oben" class="topbutton" tabindex="0">
    <toujou-icon class="icon" icon-name="arrow-up" icon-size="m" icon-color="primary"></toujou-icon>
</toujou-topbutton>
```
