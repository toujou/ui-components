# TOUJOU-Accordion

The `<toujou-accordion>` renders a customisable accordion element

## How to install

1. install via NPM with `npm install @toujou/toujou-accordion`
2. import
    1. in html with `<script src="node_modules/@toujou/toujou-accordion/dist/toujou-accordion.js"></script>`
    2. in js with: `import ToujouAccordion from '@toujou/toujou-accordion';`

## Dependencies
- Lit [^2.0.0-rc.2]
