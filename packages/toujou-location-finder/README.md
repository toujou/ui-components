# TOUJOU-LOCATION-FINDER

The `<toujou-location-finder>` an interactive map element with a list of available location both on a sidebar and as "pins" on the map.

## Dependencies
- Lit [^2.0.0-rc.2]
- @mapbox-gl [^1.13.1]
- @mapbox/mapbox-gl-geocoder [^4.7.1]
- @mapbox/mapbox-gl-language [^1.0.0]
- redux [^4.0.5]
- redux-thunk [^2.3.0]
- reselect [^4.0.0]
