# Toujou Timeline

Web-components for the toujou-timeline and toujou-timeline-legend, both used on the "Timeline" content element

The `toujou-timeline` web-component is responsible for observing which timeline elements are visible.
The `toujou-timeline-legend` web-component is responsible for updating the state of the legend active elements and for scrolling to the correct timeline element.

## Installation

      npm install @toujou/toujou-timeline
